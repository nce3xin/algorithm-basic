# 排列组合问题

## 指数型枚举

- 92 递归实现指数型枚举。从1到n这n个整数中随机选取任意多个，输出所有可能的选择方案。各行（不同方案）之间的顺序任意。样例：

  - 输入：3
  - 输出：
    ```

    3
    2
    2 3
    1
    1 3
    1 2
    1 2 3
    ```
- 1572 递归实现指数型枚举II。**包含重复数字**。给定一个长度为 n 的可包含重复数字的序列，从中随机选取任意多个数字，输出所有可能的选择方案。各行（不同方案）之间的顺序任意。样例：

  - 输入：[1,2,2]
  - 输出：
    ```

    1
    2
    1 2
    2 2
    1 2 2
    ```
